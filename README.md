# ADHD_Gamer (Pike Brain House / 派派脑力屋)

**[English]** | [中文](#中文)

## Introduction

**ADHD_Gamer** (internally developed as *Pike Brain House*) is a gamified cognitive training application designed for children (specifically targeting 6-year-olds). It combines fun gameplay with scientifically grounded training methods to help improve attention span, working memory, and impulse control. The app features a cute, kid-friendly "Macaron" aesthetic and a 3D "Pike" character companion.

### Key Features & Games

1.  **Schulte Grid Challenge (Attention):** 
    *   A classic focus training game. Players must find numbers in sequence (1-9, 1-16, etc.) on a grid.
    *   **ADHD Benefit**: Improves **sustained attention** and visual search efficiency. Helps children maintain focus on a specific task for longer periods, reducing distractibility.

2.  **Blind Box Memory (Working Memory):** 
    *   A memory game where players must memorize the contents of open boxes before they close, then answer questions.
    *   **ADHD Benefit**: Enhances **working memory** capacity. This "mental note-taking" ability is crucial for following multi-step instructions, mental arithmetic, and reading comprehension.

3.  **Ultraman Agility Training (Impulse Control):** \u2728 *New!*
    *   An action-based game featuring Ultraman themes. Players must tap to dodge incoming light beams while ignoring "feint" (fake) attacks. Includes a countdown and dynamic difficulty adaptation.
    *   **ADHD Benefit**: Trains **response inhibition** and reaction time. The "Go/No-Go" mechanics help children learn to pause and process information before acting, directly targeting impulsive behaviors and improving emotional regulation through positive reinforcement.

### Other Features
*   **Custom Mode:** Allows parents to adjust difficulty rules.
*   **Hall of Fame:** A digital sticker album with 3D ceramic-style figurines.
*   **Kid-Friendly UI:** Vertical layout, warm "Macaron" color palette, and full voice guidance (TTS).

### Tech Stack

*   **Language:** Kotlin
*   **UI Framework:** Jetpack Compose (Material3)
*   **Architecture:** MVVM + Single Activity
*   **Persistence:** Room Database, DataStore
*   **Audio:** Android TTS, MediaPlayer (Optimized Assets)

## 中文介绍

**ADHD_Gamer** (项目代号 *派派脑力屋*) 是一款专为儿童设计的游戏化认知训练应用。它将趣味游戏与科学训练方法相结合，旨在通过轻松愉快的方式帮助提升孩子的注意力、工作记忆和冲动控制能力。应用采用了温馨的“马卡龙”配色风格和可爱的3D“派派”角色 IP。

### 核心游戏与 ADHD 训练原理

1.  **舒尔特方格 (注意力训练):**
    *   经典的注意力训练游戏。玩家需要在网格中按顺序快速找出数字。
    *   **对 ADHD 的帮助**: 提升**持续性注意力** (Sustained Attention) 和视觉搜索能力。持续的视觉追踪训练能帮助孩子在特定任务上保持更长时间的专注，减少分心。

2.  **盲盒记忆 (工作记忆):**
    *   考验短时记忆的游戏。玩家需要记住盲盒打开时的物品，待盒子关闭后回答问题。
    *   **对 ADHD 的帮助**: 强化**工作记忆** (Working Memory)。如同大脑的“便签纸”，工作记忆对于阅读理解、数学计算和执行日常多步骤指令（如“穿鞋、拿书包、去门口”）至关重要。

3.  **奥特曼特训 (冲动控制):** \u2728 *新增!*
    *   结合奥特曼动作元素的反应力游戏。玩家需要根据光线发射的时机点击屏幕进行躲避，同时需要识别并忽略“假动作”欺骗。
    *   **对 ADHD 的帮助**: 专门设计的**反应抑制** (Response Inhibition) 训练。通过 Go/No-Go 范式（该动时动，不该动时不动），训练孩子在做出动作前进行“思考与判断”，有效改善**冲动行为**，并在高互动中提升情绪调节能力。

### 其他功能
*   **自定义模式:** 家长可自定义游戏难度（如网格大小、盲盒数量）。
*   **荣誉墙:** 闯关获得高分，解锁精美的3D陶瓷风格“荣誉”手办。
*   **儿童友好交互:** 竖屏设计，大字体，全程语音鼓励。

### 技术栈

*   **编程语言:** Kotlin
*   **UI 框架:** Jetpack Compose (Material3)
*   **架构模式:** MVVM + 单 Activity
*   **音频:** Android 原生 TTS, MediaPlayer (优化版语音资源)
